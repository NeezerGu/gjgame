# 闲云修行记（挂机仙侠原型）

一个以陪伴为核心的 PC 端挂机修仙小原型。保持浏览器窗口打开即可让角色在后台持续修炼，无需任何操作。

## 玩法与特性
- 自动修行：每秒累积修为与灵力，达到条件自动突破并消耗灵石。
- 灵石获取：突破缺少灵石时角色会外出打工，暂缓修行以赚取所需资源。
- 心境调适：心境长期修行会波动，低落时角色自动打坐恢复，影响修行效率。
- 自动存档：循环保存进度，关闭页面后再次打开即可延续修行状态。
- 修行日志：循环记录后台事件，便于观察角色成长。

## 运行方式
1. 安装任意现代浏览器（Chrome、Edge、Firefox 等）。
2. 下载本仓库后，直接双击 `index.html` 或使用本地服务器打开即可体验。

> 初版为纯前端实现，后续可接入存档、音效、成就、Steam SDK 等功能。
